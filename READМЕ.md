# Selenium UI Automation Test

## Project Description
This project demonstrates basic UI test automation using Java and Selenium WebDriver.

## Technologies Used
- Java
- Selenium WebDriver
- Maven

## What the test does
- Opens Google
- Searches for a keyword
- Submits the search

## How to run
1. Clone the repository
2. Open in IntelliJ IDEA
3. Run Main.java

## Example
Automates a simple Google search using ChromeDriver.
